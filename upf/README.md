# Unified Power Format (UPF)

This directory contains the UPF files used to define the power intent of the Ultra-Low Power IoT Sensor Node.

The UPF specification defines:

- Power domains
- Power switches
- Isolation strategies
- State retention
- Always-On logic
- Supply nets and ports

Using UPF separates power intent from RTL, allowing the same functional design to be synthesized with different power management strategies.
